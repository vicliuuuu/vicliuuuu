今天是把总体的方案给定下来了算是，在这里梳理一下，方便以后随时回顾：
研究重点: CUT3R + OWL-ViT + 昆虫仿生热图 + MoE-MLP 策略网络

1. OWL-ViT（语义定位层）
功能：将自然语言指令映射到图像中的目标位置（2D 框）。
特点：开放词汇、零样本能力、轻量化。
问题：2D 框投影到 3D 点云时可能出现错位，需要处理误差。

2. CUT3R（几何理解层）
功能：从连续 RGB 图像生成 瞬时 3D 点云，计算每个像素的 3D 坐标和相机位姿。
与 OWL-ViT 配合：将 2D 框映射到 3D 空间，提供目标的精确三维位置。

3. 昆虫仿生热图（安全反射层）
功能：基于 Time-To-Collision (TTC) 生成障碍物风险热图。
特点：模拟昆虫快速避障机制，为策略网络提供紧急反应信息。

4. MoE-MLP 策略网络（动作决策层）
输入：IMU/Pose、目标 3D 坐标、TTC 热图、指令 embedding。
网络设计：Mixture-of-Experts MLP，专家模块分别负责目标追踪和障碍物规避。
输出：动作指令，实现安全、快速到达目标。
