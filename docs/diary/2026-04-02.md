今天更新了一下想法，今天是外国熊布布：

In this work, we investigate the problem of object emergence from 3D semantic fields under open-vocabulary settings, motivated by the limitations of conventional detection-driven pipelines in dynamic and unstructured environments. Unlike prior approaches that rely on explicit object detectors to define discrete entities in 2D space, we propose a representation-driven framework that first constructs a continuous 3D geometric representation of the scene, and subsequently enriches it with semantic embeddings derived from CLIP.

Specifically, given a set of 3D points obtained from multi-view observations (or real-time reconstruction methods such as Stream3R), we establish correspondences between 3D points and image pixels to lift dense 2D semantic features into 3D space, forming a 3D semantic field where each point is associated with a continuous semantic embedding rather than a discrete class label. Building upon this representation, we explore whether object-level structures can emerge without explicit supervision, by performing clustering over joint geometric and semantic features.

Furthermore, we extend this formulation to the temporal domain by incorporating cross-frame association mechanisms (e.g., Hungarian matching), enabling the construction of temporally consistent, object-centric representations. This formulation naturally supports open-vocabulary reasoning, compositional semantics, and persistent object identity, which are difficult to achieve under traditional detection-based paradigms.

Our study aims to answer a fundamental question:

Can continuous 3D semantic representations, when combined with minimal structural assumptions, give rise to stable and semantically meaningful object abstractions?

To this end, we advocate a minimal and scalable experimental protocol that decouples representation learning from system-level complexity, laying the groundwork for future integration into embodied agents and world models.
